# AI Code Reviewer 🚀

An AI-powered full stack application that reviews code, suggests improvements, and helps developers learn better coding practices.

## Features

- AI code review
- Bug detection
- Code optimization suggestions
- Simple explanations
- GitHub repo support (coming soon)

## Tech Stack

- Frontend: React
- Backend: Node.js, Express
- AI: OpenAI API
- Database: MongoDB (planned)
